# Evojam Task

The task is to building a directory browser that user can explore the all folders and files inside directories.

## Live Link

The live preview of the task is [here](https://evojam.netlify.app)

## Technologies

- [ReactJs] -F
- Tailwind
- React Router
- Axios

First Install dependencies:

```sh
npm i
```

Second Start the :

```sh
npm start
```
